Work Experience

HTML
HTML is the standard markup language used to build the structure of web pages. It defines headings, paragraphs, links, images, and more—acting as the backbone of every website by organizing content in a way that browsers can interpret and display.
CSS
CSS is a style sheet language that controls how web pages look and feel. From colors and fonts to layouts and animations, CSS lets developers design beautiful and responsive interfaces that work well across various screen sizes and devices.
MongoDB
MongoDB is a document-based NoSQL database used to store data in a flexible, JSON-like format. It’s known for handling large volumes of unstructured data and is widely used in modern applications for its speed and scalability.
Express.js
Express.js is a lightweight and fast Node.js framework used to build server-side applications and APIs. It simplifies routing, middleware handling, and more, making back-end development quicker.
React.js
React.js is a popular JavaScript library developed by Facebook for building interactive user interfaces. It uses a virtual DOM and component-based structure to deliver fast, dynamic web apps with reusable, modular code.
Node.js
Node.js is a powerful JavaScript runtime built on Chrome’s V8 engine. It allows developers to write server-side code in JavaScript, enabling fast and scalable network applications ideal for real-time data processing and modern web apps.
